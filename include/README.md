# Header Files & Configuration

This directory contains the public headers for the Digital Scale modules:
- `Config.h`: Central configuration (pins, settings, filter thresholds).
- `Logger.h`: Configurable leveled logging.
- `TerminalCLI.h`: Modular Command Line Interface parser.
- `ScaleDriver.h`: HX711 hardware interaction and signal filtering.
- `ScaleModule.h`: CLI binding and state logic for the scale.
